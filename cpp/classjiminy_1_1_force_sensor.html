void jiminy::ForceSensor::reset ( void  )
overridevirtual

Reset the internal state of the sensor.

This method resets the internal state of the sensor and unset the configuration of the telemetry.

Remarks
This method is not intended to be called manually. The Model to which the sensor is added is taking care of it when its own reset method is called.
